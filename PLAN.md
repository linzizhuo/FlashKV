# FlashKV 项目计划书

> 一个基于 C++ 的轻量级 KV 存储引擎

---

## 一、项目概述

### 1.1 项目定位

基于 C++ 实现的内存键值存储引擎，提供高性能的 KV 读写与有序集合查询能力。

### 1.2 核心目标

- [ ] **从零实现哈希表** — 链地址法 + 渐进式 rehash，理解扩容对延迟的影响
- [ ] **从零实现跳表** — 随机层高 + 范围查询，理解概率平衡与红黑树的取舍
- [ ] **基于 epoll 的事件驱动网络** — 单线程事件循环，理解 Reactor 模式
- [ ] **数据结构的内存布局设计与优化** — 每个结构体以"内存效率 + 访问模式"为出发点做设计

### 1.3 技术选型

| 维度 | 选择 | 理由 |
|------|------|------|
| 语言 | C++ | 模板泛型 + RAII 安全，核心数据结构仍手写 |
| 存储模型 | 纯内存 | 极致读写性能，适合缓存场景 |
| 网络模型 | epoll（单线程事件循环） | Linux 高性能 I/O 多路复用 |
| 构建系统 | Makefile | 轻量直接，适合本项目规模 |
